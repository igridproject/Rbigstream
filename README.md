# Rbigstream
R package for easy call BigStream API.
[updated 2018-05-24]


## Rbigstream package installation


* install devtools 
```{r}
install.packages("devtools")
```

* install lastest version Rbigstream from github
```{r}
devtools::install_github("igridproject/Rbigstream")
```

## Help document
```{r}
library(Rbigstream)
?bs.connect
?storage.list
?storage.get
?storage.read
?storage.put
```
